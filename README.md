🧮 Calculadora Simples (Terminal)

Uma calculadora simples que roda diretamente no terminal através de um arquivo executável.

📌 Funcionalidades

- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão

🚀 Como usar

1. Baixe o arquivo executável.
2. Execute o arquivo:

🪟 Windows

Clique duas vezes no ".exe"
ou pelo terminal:

main.exe

🐧 Linux

Dê permissão de execução:

chmod +x main

Depois execute:

./main

🎯 Objetivo

Projeto criado para praticar lógica de programação e desenvolvimento de aplicações para terminal.
